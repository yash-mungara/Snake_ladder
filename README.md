# 🐍 🎲 Snake and Ladder Game

A classic, interactive **Snake and Ladder** board game built entirely in C++. This is a command-line interface (CLI) game that brings the nostalgic multiplayer experience straight to your terminal.

## 🎮 How to Play

1. Run the game executable.
2. Input the names of the players.
3. Players take turns rolling a standard 6-sided die.
4. Move forward by the exact number shown on the die.
5. **Ladders** will boost you forward, but **Snakes** will slide you backward!
6. The first player to reach exactly position **100** wins the game.
